# stat-184-qmd

## About This Project
This repo contains the reproducible data analysis report for STAT 184 HW #4.3. 
It covers three analyses: airport passenger traffic trends (2020-2025), 
Monte Carlo numerical integration of the standard normal distribution and a 
comparison of plan-informed vs. generic GenAI prompting using a calcium dataset.

## Data Sources
- Airport passenger data: manually compiled from publicly available airport 
  annual reports
- Calcium dataset: provided as part of STAT 184 course materials (calcium.csv)
