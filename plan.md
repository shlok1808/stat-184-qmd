# Project Plan

## Project Plan

**Goal:** Create a reproducible data analysis report using Quarto covering 
airport traffic trends, Monte Carlo integration and GenAI prompting comparison for calcium dataset.

**Needs:** R, tidyverse, knitr, Quarto, calcium.csv dataset, airport passenger data

**Steps:**
1. Collect airport passenger data for 6 airports (2020-2025)
2. Write R code for table and line plot
3. Build Monte Carlo simulation function for normal distribution
4. Use GenAI with plan-informed vs generic prompts on calcium dataset
5. Write narrative text and reflection
6. Render to PDF
